# Fantasy Lite

A sport-agnostic, themeable Fantasy Sports web application with a rich 3D user experience.

## Features

- **Configurable Rules**: Admin interface to define sports, roster slots, and scoring rules.
- **Dynamic Theming**: Switch between themes (e.g., Default vs. Fire) which updates both UI and 3D elements.
- **3D Matchups**: Interactive 3D stadium view built with React Three Fiber.
- **League Management**: Create and manage leagues and teams.

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

1.  Navigate to the project directory:
    ```bash
    cd fantasy-lite
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

### Running the Application

Start the development server:

```bash
npm run dev
```

Open your browser and navigate to `http://localhost:5173` (or the URL shown in the terminal).

### Building for Production

To create a production build:

```bash
npm run build
```

The output will be in the `dist` directory.
